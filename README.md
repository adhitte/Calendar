# My Calendar App 

A simple desktop calendar application built with **Python** and **PyQt5**.  
This project was created as a beginner-friendly exploration of GUI development in Python.

---

## Features
- Displays a calendar widget with clickable dates.
- Shows the selected date in a label below the calendar.
- Built using PyQt5 for cross-platform desktop support.
- Lightweight and easy to run.
